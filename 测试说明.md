# 编译说明

- 需要把caculate_area.cpp中的路径改成自己的路径
- 默认PNG格式，需要JPEG需要一定更改。
- 编译命令如下
```bash
g++ caculate_area.cpp -o main -lfltk -lfltk_images
```

- 也可以直接运行./main，是example2的运行结果
